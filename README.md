# SubtitleTool
> SRT 字幕时间轴偏移与转换工具

## 功能
- 打开 SRT 字幕文件，完整展示内容并可编辑
- **时间偏移**：整体偏移任意秒数（正向/负向），实时预览第一条字幕变化
- **FPS 转换**：支持 23.976/24/25/29.97/30/50/60 帧率互转，自动重算时间戳
- **SRT 合并**：合并两个 SRT 字幕文件，自动处理时间轴不重叠的情况
- 偏移/转换后可保存为新文件

## 使用方法
```bash
python subtitle_tool.py
```
打开 SRT 文件后，可执行时间偏移、FPS 转换或加载第二个 SRT 进行合并。
